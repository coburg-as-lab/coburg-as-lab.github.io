# Autonomous Systems Lab — Hochschule Coburg

Website for the Autonomous Systems Lab (ASL) at Hochschule Coburg, hosted at [coburg-as-lab.github.io](https://coburg-as-lab.github.io).

## About

The ASL operates an 8 × 8 m indoor model city at 1:6 scale for testing and validating autonomous driving algorithms with real physics, real sensors, and zero risk. The lab also runs **LUCCA**, a full-size instrumented VW eGolf with 5G connectivity and multi-sensor perception.

Research focus areas:
- **Sim2Real Transfer** — closing the gap between simulation and physical deployment
- **Adversarial Safety** — robustness testing against sensor attacks and adversarial patches
- **Planner Validation** — real-vehicle testing beyond open-loop replays
